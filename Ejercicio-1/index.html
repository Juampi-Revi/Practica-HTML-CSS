HTML

HTML son las siglas de HyperText Markup Language, que puede traducirse como el lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de una navegador a otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suele tener nombre similar.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creo la primera pagina web.

Versiones de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991.
En la actualidad, la ultima version de HTML es HTML5.
